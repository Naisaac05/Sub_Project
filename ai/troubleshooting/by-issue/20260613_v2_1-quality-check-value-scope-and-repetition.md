# v2.1 카드 품질 검사 영문 비율 범위와 오답 근거 반복

- 발생 일시: 2026-06-13
- 영역: ai
- 심각도: low

## 증상
`java-arraylist` 카드의 오답 근거 반복률이 33.3%로 정책 제한 30%를 넘었고, 영문 비율 검사가 한국어 payload에도 100%를 보고했다.

## 원인
오답 근거 두 문장이 동일한 비교 표현을 반복했다. 영문 비율 검사는 생성 내용뿐 아니라 영문 JSON 필드명까지 합친 문자열을 검사해 거짓 양성을 만들었다.

## 해결 방법
`ai/app/knowledge/concepts_v2/java/java-arraylist.json:38`의 동기화 오답 근거를 경쟁 상태와 조회 복잡도의 차이로 구체화했다. 품질 검사는 payload 값만 재귀적으로 모아 영문 비율을 계산하도록 실행 절차를 수정했다.

## 재발 방지 / 메모
오답 근거 반복률은 보기별 고유 원인을 기준으로 검사하고, 언어 비율은 스키마 키를 제외한 사용자 노출 값에만 적용한다.
