# TextInput

## 핵심 설명
TextInput의 정의는 앱 내에서 사용자가 입력하는 데이터를 관리하는 컴포넌트입니다.  
예를 들어, 로그인 사용자와 앱 전체 설정처럼 여러 화면이 공유하는 값은 TextInput의 정의에 적합합니다.

## 대표 해결
- 사용자 입력을 받기 위해 TextInput 컴포넌트를 정의하고 관리한다.
- 입력값 변경 이벤트를 처리하여 상태(State)에 값을 반영한다.

## 흔한 오해
- TextInput이 입력한 데이터를 자동으로 영구 저장(DB 등)해 주는 것은 아니다.
- 여러 화면에서 동시에 값을 공유하기 위해서는 별도의 상태 관리 라이브러리나 컨텍스트가 필요하다.
