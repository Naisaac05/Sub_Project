# v2.1.5 간접 정답 출력 예시 탐지 누락

- 발생 일시: 2026-06-13
- 영역: ai
- 심각도: medium

## 증상
`print("정답")`은 가짜 예시로 탐지했지만, 정답 문자열을 반환하는 `check_concept()`를 `print(check_concept())`로 출력하는 예시는 정상 예시처럼 점수화될 수 있었다.

## 원인
`ai/app/scripts/prepare_payload_batch_v215.py:81`의 탐지식이 출력 함수 인자 안에 정답 관련 문자열이 직접 있는 경우만 검사했다.

## 해결 방법
정답 문자열만 반환하는 함수와 해당 함수를 출력하는 호출 조합도 `print_answer`로 탐지하도록 보강했다. `ai/tests/test_prepare_payload_batch_v215.py:29`에 간접 출력 회귀 테스트를 추가했다.

## 재발 방지 / 메모
예시 품질 검사는 출력 함수 존재만 보는 대신, 코드가 실제 상태 변화와 런타임 행동을 검증하는지 함께 평가한다.
